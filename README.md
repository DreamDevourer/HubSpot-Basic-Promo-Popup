# HubSpot Basic Promo Popup
Simple and resourceful Promotional Pop-up for your HubSpot CMS. This was a browser cookie study using boolean values with jQuery fancybox to translate in days after the user closed the pop-up, making a countdown until the pop-up starts appearing again.
<br>
## How it would look
This module also supports HubSpot built-in CTA and styling options.
